Mirage2004 32x HD texture pack (bundled, optional).

Drop 32x32 PNGs here using the SAME file names as the base 16x textures
(e.g. small_tile_white.png). They override the 16x base when the player enables
"Mirage2004 32x HD textures" in Options > Resource Packs.

The generated models/blockstates are resolution-agnostic (they reference
mirage2004:block/<name>), so no model changes are needed — only the PNGs here.

Registered via AddPackFindersEvent in Mirage2004Client. Delete this README once
real textures are added (it is harmless either way).
